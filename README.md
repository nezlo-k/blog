# blog.nezlok.com

Personal blog. Leadership, technology, and whatever else is worth writing down.

Built with [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) on Jekyll, hosted on GitHub Pages.

## Writing

Posts go in `_posts/` as `YYYY-MM-DD-slug.md` with front matter:

```yaml
---
title: Your title here
date: YYYY-MM-DD
categories: [Category]
tags: [tag1, tag2]
---
```

Push to `main` → auto-deploys in ~60 seconds.
